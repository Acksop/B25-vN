iii. Les caractères spéciaux

L'utilisation d'une syntaxe entraîne l'utilisation dite de "caractères spéciaux". Ces caractères sont pour le WCode par exemple [ et ] - Mais parfois, vous pouvez avoir besoin de ces caractères ! Alors si par exemple vous devez faire une citation de Kim[62], faites comme ceci : [citation=Kim\[62\]]...[/citation]


iv. L'éditeur

L'éditeur est un outil puissant pour utiliser le WCode ! Voici les choses bonnes à savoir pour l'utiliser de manière optimale :

1 - Pour les textes ( Citation & Code ), utilisez les panneau d'insertion ! Le script se chargera d'insérer le code en remplaçant les caractères spéciaux.

2 - Pour la mise en forme basique, sélectionnez le texte sur lequel vous souhaitez appliquer la mise en forme, et cliquez sur le bouton désiré.
